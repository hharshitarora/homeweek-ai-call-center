# Terraform Infrastructure for Homeweek AI Call Center

This directory contains Terraform configuration for managing the Supabase project for Homeweek AI Call Center.

## Prerequisites

- Terraform installed
- Supabase account and Access Token

## Getting Started

1. Initialize Terraform:
   ```bash
   terraform init
   ```

2. Create a `terraform.tfvars` file with your variables or set them via environment variables.

3. Plan the changes:
   ```bash
   terraform plan
   ```

4. Apply the changes:
   ```bash
   terraform apply
   ```
